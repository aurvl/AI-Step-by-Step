# EN 
## Quick Overview

This repository contains all the resources, code examples, and notebooks associated with the **AI From Scratch** blog series — a progressive, accessible introduction to artificial intelligence for everyone.

Each directory corresponds to a blog post in the series. You'll find the code, data, and notebooks needed to run the examples yourself.

```
├── bp1-linear-regression/
├── bp2-logistic-regression/
├── bp3-neural-networks/
├── bp4-cnn-rnn/
├── bp5-transformers/
├── bp6-llms-and-prompting/
├── bp7-reasoning/
└── bp8-agents/
```

---

## Context

In 2026, AI tools are everywhere — in your inbox, your bank, your hospital, your car. Most people use them daily without understanding how they work. That gap matters.
Not because everyone needs to become an engineer. But because understanding the basics changes the way you interact with these tools — and the way you think about the world they're building around us.
This series was written for everyone: the 14-year-old who's curious, the professional who wants to use AI more effectively, the developer who wants to go deeper. No PhD required.

---

## How it unfolds
The series follows a single thread: how do you teach a machine to make decisions?

| # | Article | Core idea |
|---|---------|-----------|
| BP1 | Deciding with numbers | Linear regression — aggregating signals into a value |
| BP2 | Yes or No, with nuance | Logistic regression — turning a score into a probability |
| BP3 | One layer is not enough | Neural networks — stacking bricks to capture complexity |
| BP4 | Eyes and memory | CNNs and RNNs — shape of data, shape of the network |
| BP5 | The machine that reads | Transformers and attention — context is everything |
| BP6 | The machine that talks | LLMs — how to interact effectively |
| BP7 | The machine that reasons | Chain-of-thought, emergence, and limits |
| BP8 | The machine that acts | Agents, tools, real-world applications |

Each article builds on the previous one. The complexity grows — but the core intuition stays the same: **y = f(X)**.

### Goal

By the end of this series, you should be able to:

- understand how the AI tools you use every day actually work
- interact with them more effectively and intentionally
- read about new models and architectures without feeling lost
- build simple models yourself if you want to go further
